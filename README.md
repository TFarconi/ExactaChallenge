#Projeto para Controle de Gastos
###Descrição
Projeto desenvolvido com intuito de gerenciar os gastos de uma ou mais pessoas.

###Tecnologias utilizadas
* Java 11
* JPA
* Hibernate
* Spring Boot/Cloud/Web
* Lombok
* Netflix Eureka/Zuul
* Angular 9
* Angular Material
* Mockito/JUnit
* Database MySql
